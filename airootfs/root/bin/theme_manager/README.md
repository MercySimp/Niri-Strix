# Theme Manager for Niri Strix

A PyQt6 GUI application for managing themes across multiple applications.

## Installation

```bash
pip install PyQt6
```

## Running

```bash
cd theme_manager
python main.py
```

## Features

- Create and edit themes with color pickers
- Apply themes to multiple applications at once
- Import themes from existing config files
- Visual color preview

## Supported Applications

- btop
- kitty
- niri  
- nvim
- Waybar
- Superfile
- rofi
- dunst

## Usage

1. **Import existing theme**: Click "Import from Files", enter ex. "aetheria"
2. **Create new theme**: Click "Create New Theme", add colors, save
3. **Apply theme**: Select theme, choose apps, click "Apply Theme"
4. **Edit theme**: Select theme, click "Edit Theme", modify colors
